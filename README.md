# 🛒 E-commerce Sales Analysis Dashboard (Power BI)

## 📌 Project Overview
This project is an end-to-end E-commerce Sales Analysis dashboard built using Power BI, and DAX.  
The goal is to analyze sales performance, customer behavior, and product trends.

## 🎯 Business Problem
The company wants to:
- Track total Revenue and profit
- Analyze customer purchasing behavior
- Identify monthly trends
- track revenue by city


## 📊 Key KPIs Used
- Total Revenue
- Total Profit
- Total Retun Rate
- Total Profit Margin
- Total Net Revenue
- Customer Rating Analysis

## 🛠 Tools & Skills Used
- Power BI (Data Modeling & Visualization)
- DAX (Date Table , measures)
- Power Query (Data Cleaning)
- Data Analysis

## 📈 Key Insights
- Identified top-selling product categories
- Seasonal sales trends using time intelligence
- Customer rating performance analysis
